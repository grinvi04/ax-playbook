# 로드맵 가설 v1

> ⚠️ 입사 전 가설. 첫 달 리스닝 투어가 우선순위를 반드시 흔든다. 나침반일 뿐.

## 배경 — 제네릭 예시 시나리오

> 방법을 보여주기 위한 일반화된 예시(특정 회사 아님). 실제 입사 회사의 프로파일·주요업무로 갈아끼운다.

- 타깃 회사 = 성장기 D2C/커머스 기업 (다채널 판매, 급성장)
- **비즈니스 플랫폼팀 신설** — 재고·재무·마케팅·업무자동화 4대 영역 통합 플랫폼
- 개발팀 신설, 소수 인원 입사, 개발환경 미비
- Priority = **SCM/물류 플랫폼** (예시)

### 4대 주요업무 (예시)
1. SCM/물류 고도화 (Priority) — 통합 재고 트래킹, 유통기한/FIFO, 수요예측·자동발주(ROP)
2. 마케팅 + 재무 손익 통합 파이프라인 — 이커머스·광고 플랫폼 API, 다차원 P&L(공헌이익)
3. 전사 인사이트·모니터링 — 이상징후 감지, 의사결정 지원 시뮬
4. 전사 업무 자동화 — 문서생성(발주→인보이스 등), 워크플로 최적화

### 성장기 회사의 흔한 현실 (일반 패턴)
- 핵심 데이터를 스프레드시트로 관리 → 실질 DB이자 병목
- 비개발자가 AI로 자체 툴 제작 → 풀뿌리 AI 수요 + 섀도 IT
- 단순 반복업무로 야근

## 회사(플랫폼팀) 로드맵 — 5단계

### Phase 0 · 착지 & 최소 기반 (0~1개월)
- 최소 개발환경: 버전관리·클라우드(AWS)·시크릿·데이터 접근권한·git flow. 딱 여기까지.
- 리스닝 투어: 마케터·오퍼레이션 인터뷰, 시트 인벤토리, 섀도툴 인벤토리
- 산출물: 통증 지도 + 데이터 지도 + 첫 빠른 승리 후보 1개

### Phase 1 · 빠른 승리 & 신뢰 (1~3개월)
- 야근 유발 단순반복 2~3개 제거 (Apps Script+Claude 급, 사람 확인 원칙)
- 섀도 이메일 툴 흡수 → 1호 사례
- 자동화 요청 접수 창구 정착
- 산출물: 체감 성과 + 몰려오는 요청

### Phase 2 · 데이터 백본 (3~6개월)
- 재고 통합(국내창고·In-transit·3PL·FBA) → 단일 진실 소스, 시트는 뷰/입력창 유지
- P&L 파이프라인 착수 → 여기서 "정의 충돌"이 물리적으로 터짐
- **자동 데이터 생성 패턴**: 발주(PO)·판매 이벤트 확정 → 백본에 프로젝트/SKU 데이터 자동 생성 (거래 확정 → 데이터 자동생성)
- 산출물: 재고 SoT + 첫 진짜 대시보드

### Phase 3 · Semantic Layer / LLM 위키 + 에이전트 (6~12개월)  ← 중장기 OKF 야망 착지점
- 지표 사전(공헌이익·ACOS·ROP·FIFO)을 OKF-호환 마크다운으로, 대시보드+에이전트가 같은 정의 참조
- **MCP로 노출 → 사내 에이전트가 백본/위키에 접근**
- **슬랙에 에이전트 임베드** → 현황 조회·이상징후 요약·발주 후보 추천 (최종 판단은 인간)
- 타이밍: 아픔을 겪은 뒤 깔아야 산다

### Phase 4 · 시민개발 포장도로 & AX (12개월+)
- 수평 primitive 4~5개(우리데이터에 물어보기·리포트생성·이상알림·문서생성) + 가드레일
- 워크플로 에이전트화(이상감지·ROP 발주·PO→인보이스·판가 시뮬)
- 거버넌스 성숙(재무 숫자 환각 방지, 접근권한)

## 개인 로드맵 (회사에 겹쳐서)

- **입사 전(지금)**: 기술 과잉준비 금지. 멘탈모델 + 장난감 하나(Apps Script+Claude). 도구상자 체득.
- **Phase 0~1**: 무기 = 이미 Claude Code로 에이전틱 개발 경험. "통증→자동화를 가장 빠르게 바꾸는 사람"으로 포지셔닝.
- **Phase 3~**: 위키/semantic layer의 자연스러운 오너. 개인 memory/decisions.md 방식의 팀 스케일 확장 = 차별 베팅.

## 업무 자동화 공통 뼈대 (외워둘 것)

```
트리거 → 컨텍스트 수집 → LLM 판단 → 행동/출력 → (사람 확인)
```

- 트리거: 스케줄 · 시트 새 행 · 슬랙 명령 · 이메일 도착 · 폼 제출
- 도구 1순위: **Google Apps Script + Claude API** (시트에 바인딩, 인프라 0, JS)
- 확장(나중): n8n · 슬랙봇 · Retool
