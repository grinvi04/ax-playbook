# 첫 자동화 — 시트 문의 → Claude 영문 답변 초안

30분짜리 "hello world". 이걸 손으로 한 번 만들면 업무 자동화 도구상자(트리거·컨텍스트·LLM 호출·출력·API 키 관리)가 통째로 몸에 붙는다.

## 무엇을 만드나

구글 시트 A열에 고객 문의를 붙여넣고 → 메뉴 버튼을 누르면 → Claude가 B열에 정중한 영문 답변 초안을 써준다. **발송은 사람이** (초안까지만).

## 필요한 것

- 구글 계정 (구글 시트)
- Anthropic API 키 — https://console.anthropic.com → API Keys에서 발급 (`sk-ant-...`)

## 설정 단계 (약 10분)

1. **새 구글 시트**를 만든다.
   - A1에 `문의`, B1에 `답변 초안` 정도 헤더를 넣어두면 보기 편하다(선택).

2. **Apps Script 편집기 열기**: 시트 상단 메뉴 `확장 프로그램 → Apps Script`.

3. **코드 붙여넣기**: 편집기의 `Code.gs` 내용을 지우고, 이 폴더의 [`Code.gs`](./Code.gs) 전체를 붙여넣고 저장(💾).

4. **API 키 등록** (하드코딩 금지, 스크립트 속성에 저장):
   - Apps Script 편집기 왼쪽 `⚙️ 프로젝트 설정`.
   - 아래 `스크립트 속성` → `속성 추가`.
   - 속성: `ANTHROPIC_API_KEY` / 값: 발급받은 `sk-ant-...` 키 → 저장.

5. **메뉴 활성화**: 시트로 돌아가 새로고침(F5). 잠시 뒤 상단에 `🤖 Claude` 메뉴가 뜬다.
   - 처음 실행 시 구글이 권한 승인을 요구한다 → 본인 계정으로 승인(외부 API 호출·시트 편집 권한).

## 사용

1. A열에 고객 문의를 붙여넣는다 (여러 행 가능).
2. 해당 행을 드래그로 선택.
3. `🤖 Claude → 선택 행 답변 초안 생성` 클릭.
4. B열에 초안 생성 → 검토·수정 후 발송.

## 실험해볼 것 (여기서 실력이 는다)

- **트리거 바꾸기**: `onOpen`(버튼) 대신 `onEdit`(행 추가 시 자동)으로 → "붙여넣으면 즉시 초안".
- **컨텍스트 늘리기**: system 프롬프트에 브랜드 톤·자주 쓰는 문구·환불 정책을 넣기 → 이게 나중의 "사내 위키/semantic layer"의 씨앗.
- **모델 바꾸기**: `Code.gs`의 `MODEL` 한 줄을 `claude-haiku-4-5`(저렴·빠름) 또는 `claude-sonnet-5`로.
- **출력 바꾸기**: 옆 셀 대신 Gmail 초안 생성(`GmailApp.createDraft`)으로 → "문의 → 이메일 초안까지".

## 배운 것 정리 (뼈대 재확인)

| 뼈대 단계 | 이 코드에서 |
|---|---|
| 트리거 | `onOpen` 커스텀 메뉴 버튼 |
| 컨텍스트 수집 | `sheet.getRange(row, 1).getValue()` (A열 문의) |
| LLM 판단 | `callClaude()` → `UrlFetchApp`로 Messages API |
| 행동/출력 | `draftCell.setValue()` (B열 초안) |
| 사람 확인 | 초안까지만, 자동 발송 없음 |

이 문장 하나의 변주가 앞으로 만들 자동화의 80%다:
**트리거 → 컨텍스트 → LLM 판단 → 출력 → (사람 확인)**
